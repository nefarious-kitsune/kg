Don't reveal Haka Boss too soon

If you have the Cave Map and can see everything under slates, leave Haka Boss as the last thing you uncover. When you uncover something, all neighboring slates become unlocked.

Watch the Gate

Sometimes the path to the Gate is blocked by many Boulders and Haka Boss is defeated before the Gate is uncovered. If you see that there is no red cross sign on the Gate, it means the Gate is already open, so proceed to the next stage.
