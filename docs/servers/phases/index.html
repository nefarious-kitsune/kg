Server Phases

A server has three main phases:

  • The Beginning Phase: the server starts with a fog which gradually lift away as various alliances expand their territories. Ultimately the alliances will fight for the throne and a King/Queen will be selected after the war.
  • The SvS Phase: the server will compete with other servers the 3-Way Cross-Server Invasion as well as other cross-server events.
  • The KvK Phase: an active alliances will join force with other alliances in the 4-way Burning Expedition.
