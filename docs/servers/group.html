Server Cluster

There are three main groups of Kingdom Guard servers, which many players refer to as S1, S2, and S3 servers.

The first cluster (S1) include all servers numbered 9001 to 9024. The entire group was locked shortly after the second group was opened.

The second cluster (S2) include all servers numbered 401 to 1150 as well as 2002. As of Season 27, the group has around 360 active servers (see Server Status).

The third cluster (S3) included all servers numbered 1151 or above. New servers in this cluster are opened on a daily basis (see S3 opening history).

Server Differences

S2 and S3 servers have different configurations. Because of this, transfer between the two groups is currently impossible.
