Forest Spirit Prayer Festival

2023-10-11 ~ 2023-10-19.

Event Rewards

  • Forest Spirit Outpost castle skin
  • Forest Spirit King Court castle skin
  • Tree of Life avatar frame
  • Rila - Forest Spirit Guardian

Mini Events

How to Play:

The player complete Event Tasks (Hospitality of the Forest Elder) and Daily Tasks (Clay Collection) to earn Level-1 Token (Clay).

The player use the Level-1 Token (Clay) to play the puzzle game (Pray and Play Elimination) and receive points.

The player use the points earned to redeem Level-2 Token (Life Essence) in Point Rewards (Elder's Reciprocation).

The player participate in Wishing Pool (Tree of Life) to earn Level-2 Token (Life Essence).

The player use the Level-2 Token (Life Essence) to do Lucky Draws (Sacred Tree Secret Cave).
