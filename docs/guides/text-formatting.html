Text Formatting

You can format text message with simple code:

  • <b>text</b> will set the text in bold.
  • <i>text</i> will set the text in italic.
  • <size=40>text</size> will set the text size bigger.
  • <color=red>text</color> will set the text in red color.
    black silver gray white maroon red purple fuchsia green lime olive yellow navy blue teal aqua
  • Formatting tags must be closed properly. For example, <color=red><size=40>text</color></size> will not display properly.
