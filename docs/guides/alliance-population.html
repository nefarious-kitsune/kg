Alliance Population

Information on how alliance population (member count) limits are calculated.

Sheets-based calculator is available.

  • An alliance has a base population limit of 50.

  • The alliance can raise the limit by occupying Dragon Nests

    • Each level-1 Dragon Nest raises the limit by 2.
    • Each level-2 Dragon Nest raises the limit by 3.
    • Each level-3 Dragon Nest raises the limit by 4.
    • Each level-4 Dragon Nest raises the limit by 5.
    • Each level-5 Dragon Nest raises the limit by 6.
    • The maximum limit increase from Dragon Nests is 20.

  • The alliance can raise the limit by Alliance Tech, and the maximum limit increase from Alliance Tech is 30.

  • The alliance can raise the limit by earning Glory Stars from Burning Expedition victories.

    Each Glory Star raises the limit by 10, and the maximum limit increase from Glory Stars is 30.

  • The maximum population limit an alliance can possibly have is 130.
