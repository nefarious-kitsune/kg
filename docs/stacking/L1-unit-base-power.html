Unit Base Power

Each troops march can have a certain number of tower defense units. These units contribute to the Unit Base Power.

The Unit Base Power depends on Castle level, Troops capacity, and Unit Specialization.

Calculation

Unit Base Power = 10× Castle Level × Troops Capacity × ( Unit Spec. bonus + 100% )

Stacking

Troops capacity

By default, each troops march has six (6) units. The unit count can be increased by four (4) via Academy Tech. The unit count can be increased by nine (9) via Expedition Tech.

Example

Example
Castle Level 810
Troops Capacity 9
Unit Specialization (Archer) +0%
Unit Specialization (Fire) +10%
Unit Specialization (Ice) +20%
Unit Specialization (Goblin) +20%
Unit Base Power (Archer)
8100 × 9 × (1 + 10% + 20% + 20%) 		1093500
Unit Base Power (Fire)
8100 × 9 × (1+ 0% + 20% + 20%) 		1020600
Unit Base Power (Ice)
8100 × 9 × (1 + 0% + 10% + 20%) 		947700
Unit Base Power (Goblin)
8100 × 9 × (1 + 0% + 10% + 20%) 		947700
