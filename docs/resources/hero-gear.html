Hero Gear

Information on how to acquire Hero Gear (Weapon).

Recommendation

Collect T5 gears from Evil Spirit Invasion and participate in the Mightiest Kingdom event for higher-tier gears.

Availability

Tier How to acquire
T2 Formerly available through events
T3 Formerly available through events
T4 Formerly available through events
T5 Available as damage reward in Evil Spirit Invasion
T6 Available as daily reward in the Mightiest Kingdom event
T7 Available as rank reward for the Personal Scoreboard in the Burning Expedition event.
T8 Available as rank reward in the Mightiest Kingdom event and as rank reward for the Personal Scoreboard in the Burning Expedition event.
T9 Available as rank reward in the Mightiest Kingdom event
